# Landing_Page_AB_Test_Statistical_Analysis

## Overview
This project presents a comprehensive statistical analysis of an A/B experiment conducted on a landing page. The objective is to evaluate performance differences between control and treatment groups and determine whether observed variations are statistically significant.

## Objectives
- Compare conversion rates between Group A (Control) and Group B (Treatment)
- Apply statistical hypothesis testing (t-test and Chi-square test)
- Perform segmentation analysis (e.g., by region, device, or traffic source)
- Provide data-driven recommendations based on findings

## Methodology
The analysis follows a structured experimental framework:

1. Data cleaning and preprocessing
2. Creation of balanced experimental groups
3. Conversion rate calculation
4. Statistical hypothesis testing:
   - Independent t-test (mean comparison)
   - Chi-square test (categorical association)
5. Segment-level performance evaluation
6. Interpretation and business insights

## Tools & Technologies
- Python
- Pandas
- SciPy
- Matplotlib
- Seaborn

## Key Insights
The results identify whether the treatment version significantly improves conversion performance and highlight behavioral differences across segments. Findings are interpreted from both statistical and business perspectives.

## Conclusion
This project demonstrates the practical application of statistical inference in real-world digital experimentation, supporting data-driven decision-making for product optimization.
